Intro
===========================

Overview
------------

JRI Publisher installs and configures Tomcat, Java, and JasperReportsIntegration.  

It also provides a module for Publishing, Scheduling, and Managing Reports. ::


System Requirements
-------------------
CentOS 8

Ubuntu 18

Webmin

Postfix
